## Number Hunt
A simple Python GUI game where you guess a random number between 1 and 50.
How to Play
1. Enter a number between 1 and 50 in the input box.
2. Click **Guess** to check your number.
3. The game will tell you if your guess is too high, too low, or correct.
4. Use the **Hint** button to see if the number is even or odd.
5. Click **Restart** to play again or **Quit** to exit.

## Features
- Graphical interface using Tkinter
- Random number generation
- Attempt counter
- Hint button for even/odd

## How to Run
1. Make sure Python is installed.
2. Save the code.
3. And run the file.
